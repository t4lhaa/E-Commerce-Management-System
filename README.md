## 🛒 E-Commerce Management System
This project is a console-based **E-Commerce Management System** developed in the **C programming language**. It is designed to simulate the core operations of an online shopping platform by providing separate **Admin** and **Customer** panels. The system enables efficient **inventory management**, **shopping cart handling**, **order processing**, and **billing**, while also maintaining a record of total sales. Built using structured and modular programming concepts, this project demonstrates practical use of functions, arrays, structures, and control flow to model real-world e-commerce operations in a simple and user-friendly console environment.
## 📌 Features

#### 🔐 Admin Panel
- View all available stock  
- View low-stock items  
- Restock existing items  
- Remove stock from inventory  
- Add new products  
- View total sales revenue  
- Secure logout system  

#### 👤 Customer Panel
- View available products  
- Add products to cart  
- View cart details  
- Remove items from cart  
- Finalize order with customer details  
- Auto-generated receipt  
- Logout option  

#### 🧾 Billing System
- Calculates total price of items  
- Applies stock updates in real-time  
- Generates a detailed receipt  
- Maintains overall sales record  

#### 🖥️ Screens Included
- Main Menu  
- Admin Dashboard  
- Customer Dashboard  
- Inventory Display  
- Shopping Cart  
- Final Receipt  

#### 📚 Concepts Demonstrated
- Functions and modular programming  
- Conditional logic  
- Loops  
- Arrays and structures  
- Input/output handling  
- Real-world inventory simulation
